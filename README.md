## ⚡️ 快速开始

1. 下载本仓库的代码
2. 安装 Python 依赖 `pip install opencv-python opencv-contrib-python`
3. 将需要识别的图片放置在与 `.py` 文件同级目录下的 `target` 文件夹中
4. `python scan.py`
